# Casa Duculescu

Website de prezentare pentru Pensiunea Casa Duculescu, Ocnele Mari, județul Vâlcea.

## Demo

🔗 https://stef14-bit.github.io/casa-duculescu/

## Tehnologii

- HTML5
- CSS3 (variabile CSS, design responsive, mobile-first)
- Fără framework-uri sau librării externe

## Structură

```
.
├── index.html
├── css/
│   └── style.css
├── media/
│   └── ...imagini
└── README.md
```

## Rulare locală

Deschide `index.html` direct într-un browser, sau folosește extensia **Live Server** din VS Code pentru reîncărcare automată.
